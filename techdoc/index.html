Introduction

SampleScript is a fictional programming language designed for educational purposes.

This document will help you understand the basics of SampleScript.

Variables

In SampleScript, you can declare variables using the let keyword.

Here are some examples:

let x = 10; let name = "Sample";

Variables can hold different types of data including numbers, strings, and booleans.

Functions

Functions are reusable blocks of code. You can define a function using the function keyword.

Example:

function greet() { return "Hello, world!"; }

Functions can take parameters and return values.

Control Flow

SampleScript includes various control flow statements such as if, else, and switch.

Example:

if (x > 5) { console.log("x is greater than 5"); }
  • if statements for conditional execution
  • for loops for iteration
  • while loops for repeated execution
Arrays

Arrays are used to store multiple values in a single variable. They are declared using square brackets.

Example:

let colors = ["red", "green", "blue"];
  • Access elements using their index: colors[0] returns "red"
  • Use length property to find the number of elements: colors.length
